# Papounouche Bien-Etre

Site vitrine React pour une activite de medecine douce, avec Tailwind, animations et prise de rendez-vous via un calendrier embarque.

## Lancer le projet

```bash
npm install
npm run dev
```

## Build production

```bash
npm run build
```

## Reservation

Configurez l'URL de reservation dans un fichier `.env` :

```bash
VITE_BOOKING_URL=https://calendly.com/votre-compte/decouverte
```

Le projet est actuellement prevu pour integrer rapidement Calendly ou tout agenda externe compatible iframe.
